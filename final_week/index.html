麗澤大学キャンパスマップ

このマップを作った背景と目的

既存の麗澤大学のキャンパスマップではパッと見ただけではどの建物がどこにあるのかわからず迷ってしまう学生が居たり、建物の位置が実際とはずれていたり説明と建物がバラバラで表示されていて見づらい、建物によっては説明すらないという問題があった。
そこで、より直感的に操作でき、自分がどこにいて目的地がどこにあるのかが一目でわかるような新しいキャンパスマップを麗澤大学や付属の中高が気になっている学生、麗澤大学のOB・OG、現役の麗澤大学生を主なターゲットとして制作した。

このマップの説明

麗澤大学・付属中高のほぼすべての建物を3Dポリゴンで表示し、各施設を実際の写真を使ったマーカーで表示しどの施設がどこにあるのかを分かりやすくしている。
また、過去の麗澤大学付近のマップを表示して過去と現在の敷地、建物の違いを見ることができる。
現在地から正門までの最短経路を表示する機能もあり、初めて麗澤大学に来た人でも迷わず正門まで行くことができる。
さらに、建物のジャンルごとに絞り込んだり、建物名で検索したり、マーカーをクリックすることで建物の詳細情報を表示したりすることができる。

使用データの内容・出典

  1. 麗澤大学の建物の座標データ
    ( Google Maps )
  2. 麗澤大学・付属中高の建物・施設の写真
  3. 麗澤大学の各施設の説明
    ( 麗澤大学 大学入試・広報課 )
  4. 麗澤大学の過去のマップデータ
    ( 国土地理院 , Google Maps )
